# PREPARE Phase

## Data Source
Dataset: YouTube Trending Dataset (Updated Daily)
Source: Kaggle

## Data Description
The dataset contains:
- Video title
- Channel name
- Category
- Views, likes, comments
- Publish date
- Trending date
- Tags

## Data Credibility
- Public dataset from Kaggle
- Frequently updated
- Suitable for trend analysis

## Limitations
- Region-specific bias
- Missing or inconsistent tags
- No direct revenue data

## Tools Used
- Python (Pandas, NumPy)
- Jupyter Notebook (Kaggle)
- Tableau for visualization
